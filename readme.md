# Fluent Icons Cheatsheet

A searchable, interactive cheatsheet for Microsoft's Fluent System Icons. Browse, search, and copy icon codepoints.

## Features

- **Search** - Find icons by name or codepoint
- **Theme Toggle** - Switch between light and dark modes
- **Density Options** - Choose between dense and sparse grid layouts
- **Click to Copy** - Copy icon codepoints with one click
- **Auto-Updated** - Automatically fetches the latest icons weekly
